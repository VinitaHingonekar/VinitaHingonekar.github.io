Forgotten Treasure of Evergreen

2D Platformer Game in Unity


Project Overview


"Forgotten Treasure of the Evergreen" is a 2D platformer game developed in Unity, consisting of 10 challenging levels.After completing all levels, players are rewarded with a map leading to the treasure.



Key Features


  1. Unity Systems:
    • Input System: Facilitated smooth player controls and interactions.
    • Sound System: Managed diverse sound effects for a more immersive experience.
    • Particle System: For creating visual effects
    • Animator System: Brought the game’s characters and objects to life with detailed animations.

  2. Gameplay Mechanics:
    • Player can move, jump, and throw bullets to defeat enemies.
    • 3 Different Enemies
      • Porcupines: Spiky adversaries.
      • Spearmen: Throw spears to challenge the player.
      • Spiders: Quick and dangerous, requiring swift reactions.
    • Obstacles: Players must navigate through spikes, water hazards, and spider webs that slow them down.
    • Various Interactive elements
      • Movable Boxes: Used to reach higher platforms.
      • Moving Platforms: Add complexity to the levels.
      • Lock-Key Mechanic: A door to the next level opens only after the key is collected.

  3. Score Management:
    • Implemented a score manager that tracks and stores scores separately for each level.
    • The total score is displayed after completing the game.

  4. Lives System:
    • Players have 5 lives to complete the game, adding an element of challenge and strategy.

  5. User Interface:
    • Main Menu and Levels Menu: Allows players to start the game or choose specific levels.
    • Sound Control: Option to mute and unmute the game's sound effects.
    • Level Transitions: Smooth transitions between levels enhance the gameplay experience.

  6. 2D Animations:
    • Created and animated various in-game elements, including player movements, enemy actions, and the door opening sequence when a key is collected.

  7. Parallax Background:
    • Added depth to the game by implementing a parallax scrolling background, enhancing the visual experience.

  8. Sound Effects:
    Integrated distinct sound effects for:
    • Player Actions: Jumping, shooting bullets.
    • Item Interactions: Key and coin pickups.
    • Game Events: Level completion, player death.


Visual Design:

  1. All game assets were designed using Inkscape, showcasing my skills in graphic design:
    • Player and Enemy Assets: Complete with animations for lifelike movements.
    • Jungle-Themed Tileset: Designed a full set to create the game’s environment.
    • Background Layers: To support the parallax effect.
    • Interactive Sprites: Including the box, key, and door designs.
